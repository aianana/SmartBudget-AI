# SmartBudget AI

AI-powered personal finance assistant that analyzes CSV/PDF bank statements,
categorizes expenses, detects spending patterns, and generates actionable financial insights.

## Features

- Upload bank statements (CSV/PDF)
- Automatic expense categorization
- Spending pattern analysis
- AI-generated budgeting recommendations
- Expense statistics dashboard

## Tech Stack

Frontend:
- React / Next.js

Backend:
- Python FastAPI / Node.js

AI:
- OpenAI API

DevOps:
- Docker
- GitHub Actions
